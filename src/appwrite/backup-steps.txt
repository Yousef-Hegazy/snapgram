1- Make sure you're inside appwrite folder and docker compose is stopped

2- Pull database: docker run --rm -v appwrite_appwrite-mariadb:/data -v ${PWD}:/backup ubuntu tar czf /backup/appwrite-db-backup.tar.gz -C /data .

3- Pull storage: docker run --rm -v appwrite_appwrite-uploads:/data -v ${PWD}:/backup ubuntu tar czf /backup/appwrite-uploads-backup.tar.gz -C /data .

4- Push database: docker run --rm -v appwrite_appwrite-mariadb:/data -v ${PWD}:/backup ubuntu tar xzf /backup/appwrite-db-backup.tar.gz -C /data

5- Push storage: docker run --rm -v appwrite_appwrite-uploads:/data -v ${PWD}:/backup ubuntu tar xzf /backup/appwrite-uploads-backup.tar.gz -C /data
